# Smart File Organizer 📂

智能文件整理工具 — 自动扫描目录，按文件类型分类归档。

## 功能

- 🔍 自动识别 9 大类文件（文档、图片、视频、音频、代码...）
- 👁️ 预览模式（`--dry-run`）— 先看效果再执行
- ⚙️ 支持自定义分类规则
- 📊 整理前后统计报告
- 🛡️ 自动处理重名文件，安全不覆盖

## 快速开始

```bash
# 预览模式（推荐先用）
node index.js ~/Downloads --dry-run

# 正式整理
node index.js ~/Desktop

# 使用自定义规则
node index.js ~/Documents --config ./rules.json
```
